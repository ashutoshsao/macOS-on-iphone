## Docs

This directory will contain the canonical documentation for the project:

- `boot/` – boot chain, secure boot, and dual-boot concepts.
- `hardware/` – SoC and board-level hardware mapping.
- `drivers/` – per-subsystem driver and reverse-engineering notes.
- `security/` – SEP, exclaves, and the wider trust model.
- `status/` – subsystem status matrices and progress logs.

The plan is to keep these Markdown files as the single source of truth and then mirror them into the Hugo site under `site/content/docs/` so they are readable both on GitHub and as a website.
